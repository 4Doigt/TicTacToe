# MonProjet

Il s'agit du jeu du TicTacToe alias le jeu du morpion

# Explication

J'ai prefere utilise le tableau sous forme deux dimensions je trouvais ca plus intuitif. J'ai decide d'ameliorer l'IA et de le rendre plus strategique. L'IA devait prendre le centre si il etait disponible, sinon elle devait empecher le joueur adverse de jouer un coup gagnant ou sinon elle devait jouer sur une case libre.
